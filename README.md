# MDH PDF Converter App

This is a Streamlit-based web app that allows users to convert PDF files into Word (.docx) or Excel (.xlsx) formats.

## Features
- PDF to Word conversion
- PDF to Excel conversion with table detection
- MDH branding and logo
- Help section and version history

## How to Use
1. Upload a PDF file.
2. Choose conversion type.
3. Click Convert and download the result.

## Technologies Used
- Streamlit
- PyMuPDF
- pdfplumber
- python-docx
- pandas
- openpyxl
